
```shell script
oc create -f https://raw.githubusercontent.com/ibm-garage-cloud/ibm-garage-iteration-zero/master/install/install-ibm-toolkit.yaml
sleep 5
oc wait pod -l job-name=ibm-toolkit --for=condition=Ready -n default
oc logs job/ibm-toolkit -f -n default

```

### A. Download the manifest `install-toolkit-yaml`

1. Download the latest Toolkit installation manifest yaml file
    ```shell script
    curl -LO https://raw.githubusercontent.com/ibm-garage-cloud/ibm-garage-iteration-zero/master/install/install-ibm-toolkit.yaml
    ```
1. If you want a specific version of the manifest yaml file visit the [release page](https://github.com/ibm-garage-cloud/ibm-garage-iteration-zero/releases) and download the corresponding manifest file `install-toolkit-yaml`

### B. Customize Installation

1.  Edit the file `install-toolkit-yaml` if you want to change any of the defaults, such as changing the tools namespace, adding a components from the catalog directory or removing a default component.

### C. Run the installation using the OpenShift CLI

1. Run the installation job after customizing the installation script
    ```shell script
    oc create -f install-ibm-toolkit.yaml
    sleep 5
    oc wait pod -l job-name=ibm-toolkit --for=condition=Ready -n default
    oc logs job/ibm-toolkit -f -n default

    ```
